# GEO P2 Comparison Content Map

Date: 2026-06-04
Branch: `docs/geo-p2-comparison-content-map`
Brand spelling: `Lynxtour`

## Scope

This is a documentation-only plan for P2 visible comparison and decision content. GEO P1 answer blocks have already been implemented and merged; P2 should add or refine short, practical comparison blocks that help real travelers choose a route and give AI answer systems clean extraction points.

No Liquid code, templates, sections, snippets, JSON-LD, schema, CSS, JavaScript, product prices, forms, cart, checkout, URLs, handles, canonical, robots, or content files should be changed in this documentation phase.

## Shared Guardrails

- Keep blocks short, visible, and useful for travelers.
- Do not keyword-stuff or use robotic labels.
- Do not duplicate P1 answer blocks or FAQ content.
- Do not add schema.
- Do not hide text.
- Add no more than one compact P2 comparison block per target page.
- Use only existing safe internal URLs.
- Keep the tone natural, practical, and travel-consultant-like.
