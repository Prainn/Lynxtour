# Lynxtour Codex Project Instructions

This repository is a Shopify Liquid theme for Lynxtour, an inbound Yunnan private tour agency.

Core rules:
- Make ultra-conservative changes.
- Do not redesign pages unless explicitly requested.
- Do not change URLs, handles, template names, or section names unless explicitly requested.
- Preserve Shopify section schema and existing settings.
- Prefer handle-scoped Liquid logic over broad global changes.
- For SEO work, check title, meta description, H1, canonical, robots, OG/Twitter, JSON-LD, internal links, and indexation risk.
- For product pages, preserve booking UI and variant logic.
- For social/content work, write for overseas travelers interested in Yunnan private tours.
- Always explain changed files, old logic, new logic, and rollback risk.
- Do not commit changes unless explicitly asked.
